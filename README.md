# Caesar Cipher

A simple Python program to encrypt and decrypt text using the Caesar cipher technique.

## Description

This program allows users to secure their messages by shifting letters in the alphabet by a user-defined value. It supports both encryption and decryption of text, and can handle letters and numbers.

## Features

- Encrypt text with a shift value
- Decrypt text with a shift value
- Supports letters and numbers
- User-friendly prompts for interactive use

## Technologies Used 
- Python 3.12.3
  
## Example

Encrypt or decrypt? (e/d): e
Enter text: Hello123
Enter shift value (1-25): 3
Encrypted text: Khoor123
